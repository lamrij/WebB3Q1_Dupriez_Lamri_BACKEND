definition : 
Mathis -> branch local (sur ton pc)
origin/mathis -> branch distante (sur les serveur de github)


// switch sur une autre branch qui n'existe pas encore en local
[COMMAND]
git switch -c Mathis origin/Mathis
[DETAIL]
- git switch -> passé sur une autre branch
- "-C" create local branch, "Mathis" nom de la branch local créer
- origin/Mathis, nom de la branch remote que l'on souhaite lier a cette nouvelle branch


// switch sur une branch qui existe en local
[COMMAND]
git swtich Mathis


//push sur le git (ca va push sur la branch que tu utilise)

// exemple : 
Si tu est sur la branch Mathis, lier a origin/Mathis -> ca va push sur origin/mathis le contenu de Mathis 

// 1 on ajoute toute les modifications au staging area (zone de préparation) dans le git local
[COMMAND]
git add .

// 2. on commit sur le git local
[COMMAND] 
git commit -m "ton message"

// 2. on push sur le git distant
[COMMAND]
git push
